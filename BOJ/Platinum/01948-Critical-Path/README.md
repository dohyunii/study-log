## [[BOJ-1948] 임계경로 (Critical Path)](https://www.acmicpc.net/problem/1948)

> **Topic:** Graph Theory, Graph Traversal, Directed Acyclic Graph, Topological Sorting, Traceback\
> **Difficulty:** Platinum 5

### Complexity Analysis
* **Time Complexity:** $O(N+M)$
* **Space Complexity:** $O(N+M)$

### Mistakes & Lessons
**Learned:** It is possible to find longest/shortes path using topological sort in DAG.
