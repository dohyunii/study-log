## [[BOJ-34713] Collecting Sogang](https://www.acmicpc.net/problem/34713)

> **Topic:** Binary Search, Greedy
> **Difficulty:** Platinum 3

### Complexity Analysis
* **Time Complexity:** $O(N\log(N))$
* **Space Complexity:** $O(N)$

### Mistakes & Lessons
**Lesson:** Tracking pointer for every parameter to optimize time complexity.
**Mistake:** It is not optimal to take the closest alphabet at once.
