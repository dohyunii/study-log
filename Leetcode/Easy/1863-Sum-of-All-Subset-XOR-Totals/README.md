## [[LC-1836] Sum of All Subset XOR Totals](https://leetcode.com/problems/sum-of-all-subset-xor-totals/)

> **Topic:** Bitmask, Combinatorics, Mathematics
> **Difficulty:** Easy

### Complexity Analysis
* **Time Complexity:** $O(N)$
* **Space Complexity:** $O(N)$

### Mistakes & Lessons
**Lesson:** New way to imagine about XOR operation subset. Adding one bit set to 1 to a subset flips the XOR total.
